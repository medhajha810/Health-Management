Health Management System - Troubleshooting

Having trouble loading the application?

Try these direct links to different parts of the application:

Landing Page Project Details Dashboard User Profile Records New Record FAQ Contact

Common issues and solutions:

  • Infinite loading spinner: Try the direct links above or clear your browser cache and cookies.
  • Blank page: Ensure JavaScript is enabled in your browser settings.
  • Missing elements: Try using a different browser like Chrome or Firefox.
  • Routing issues: Use the direct links above which bypass the standard navigation.

Try these steps if you're still experiencing issues:

  1. Refresh the page (F5 or Ctrl+R)
  2. Clear browser cache (Ctrl+Shift+Delete in most browsers)
  3. Try opening the app in an incognito/private browsing window
  4. Use a different browser
  5. Disable browser extensions that might interfere with the application

Last resort options:

If you still can't access the application, try these options:

  • Skip all animations
  • Minimal load mode
